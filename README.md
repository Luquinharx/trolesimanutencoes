# Trolesi Manutenções

Este é o repositório oficial da landing page da **Trolesi Manutenções**, uma empresa especialista em soluções industriais, com foco em segurança, manutenções complexas e confiabilidade estrutural para o mercado de gás de cozinha e fluidos sob pressão.

O projeto foi desenvolvido como uma Single Page Application (SPA) moderna, rápida e responsiva.

## 🚀 Tecnologias Utilizadas

- **[React](https://react.dev/)**: Biblioteca JavaScript para construção de interfaces de usuário.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build rápida e leve.
- **[TypeScript](https://www.typescriptlang.org/)**: Superset do JavaScript que adiciona tipagem estática.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para animações fluidas e interações.
- **[Lucide React](https://lucide.dev/)**: Biblioteca de ícones leve e consistente.

## 🛠️ Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e computadores.
- **Animações Fluidas**: Utilização do Framer Motion para uma experiência de usuário envolvente.
- **Seções Informativas**:
  - **Início (Hero)**: Apresentação impactante da empresa.
  - **Sobre Nós**: História e valores da Trolesi Manutenções.
  - **Serviços**: Detalhes sobre Montagem Industrial, Manutenção Preventiva e Desmontagem/Retrofit.
- **Botão Flutuante do WhatsApp**: Facilita o contato direto com a empresa.

## 📦 Como Rodar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd trolesimanutencao
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Build para produção:**
   ```bash
   npm run build
   ```
